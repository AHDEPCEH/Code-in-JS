Постановка задачи:
- Написать алгоритм кодирования rle.
- Протестировать работу алгоритма, кодировки и декодирования строки.
Процесс: В текстовом файле input.txt записана строка.Результат работы кодировщика сохранить в файле code.txt. После этого рас-
кодировать сроку из файла code.txt, результат сохранить в decode.txt. И проверяем на схожесть файлов input.txt и decode.txt

Режим работы программы (т. е. кодирование или декодиро-
вание), имя файла с входной строкой и имя файла для записи

результата задаются как аргументы командной строки. Напри-
мер:

- node rle.js code input.txt code.txt
- node rle.js decode code.txt decode.txt
- FC input.txt decode.txt
