Постановка задачи:
- При запуске файла передаётся 2 параметра: 1 - название файла, где находится строка для кодирования, 2 - название файла, куда сохранить закодированную строку.
- В консоль выводится дерево всех символов.
- Создаётся файл decode.txt, куда записывается декодирование закодированной строки.
