Постановка задачи:
На вход подается строка — последовательность букв. Тре-
буется:

1 ) построить алфавит (т. е. множество всех различных сим-
волов исходной строки) и найти частоту для всех символов ал-
фавита;

2 ) для полученной таблицы рассчитать энтропию согласно формуле.
Процесс:
node entropy.js input.txt
В качестве аргумента передаётся название файла, содержащего строку.
результат работы программы выводим в консоль.
